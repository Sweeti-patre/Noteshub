# NotesHub 📚

NotesHub is a web-based notes sharing platform developed for students and teachers. The platform allows users to upload, download, and manage study materials in one place.

## Features

- User Authentication (Login / Signup)
- Student and Teacher Roles
- Notes Upload System
- Download Notes
- Contact Support Page
- Responsive UI (Desktop + Mobile)
- Modern Glassmorphism Design
- Role Selection System
- Semester Selection
- Secure Access Control

## Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Python
- Django

**Database**
- SQLite
